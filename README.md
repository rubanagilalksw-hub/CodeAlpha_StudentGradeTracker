# CodeAlpha_StudentGradeTracker
# 🎓 Student Grade Tracker

A desktop **Java Swing** application to manage student records — add students, track grades, and view performance statistics through a clean, modern dashboard interface.

> Built as a **Java Internship Task** for **CodeAlpha**.

---

## ✨ Features

- 📊 **Dashboard** — quick overview with a live student records table
- ➕ **Add Student** — easily add new student records with name and scores
- 👥 **View Students** — browse, search, and manage all saved student entries
- 📈 **Statistics** — auto-calculated insights:
  - Total Students
  - Average Percentage
  - Highest Score
  - Lowest Score
- 📤 **Export** and **Clear All** options for record management
- 🎨 Custom rounded UI components for a modern, polished look
- ℹ️ **About** section with project info

---

## 🛠️ Tech Stack

- **Language:** Java
- **GUI Framework:** Java Swing (`javax.swing`, `java.awt`)
- **Build Tool:** Apache Ant (NetBeans project)
- **IDE used:** NetBeans / VS Code (Java Extension Pack)

---

## 📂 Project Structure

```
StudentGradeTracker/
├── src/
│   └── studentgradetracker/
│       ├── Main.java              # Application entry point
│       ├── MainFrame.java         # Main window & navigation
│       ├── DashboardPanel.java    # Dashboard view
│       ├── AddStudentPanel.java   # Add student form
│       ├── ViewStudentsPanel.java # Student records table
│       ├── StatisticsPanel.java   # Performance statistics
│       ├── AboutPanel.java        # About section
│       ├── RoundedPanel.java      # Custom rounded UI component
│       └── Student.java           # Student data model
├── build.xml                      # Ant build script
└── nbproject/                     # NetBeans project configuration
```

---

## 🚀 How to Run

### Option 1 — NetBeans
1. Open NetBeans → **File → Open Project**
2. Select the `StudentGradeTracker` folder
3. Click **Run** ▶️

### Option 2 — VS Code
1. Install **Extension Pack for Java** (Microsoft)
2. **File → Open Folder** → select the `StudentGradeTracker` folder
3. Open `src/studentgradetracker/Main.java`
4. Click the ▶️ **Run** button above the `main` method
